# AddyCaddyClient

A minimal helper for FourSquare API requests using an address

## Installation

Add this line to your application's Gemfile:

    gem 'addy_caddy_client'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install addy_caddy_client
